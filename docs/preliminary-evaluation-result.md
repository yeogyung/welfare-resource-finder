# 찾아봇 1차 내부 평가 결과

생성일: 2026-06-14

## 평가 범위

본 평가는 최종 MVP에 내장된 `418건` 복지자원 DB와 DB 기반 추천 엔진을 대상으로 수행한 1차 내부 평가이다. 사용자 화면에는 평가 지표를 노출하지 않고, `/admin.html`의 내부 평가 화면과 동일한 15개 질문 시나리오를 기준으로 검색·추천 결과의 적합성을 확인하였다.

## 수행한 평가

| 항목 | 내용 | 결과 |
|---|---|---:|
| 평가 문항 수 | 어르신 질문 시나리오 | 15개 |
| Top-3 적합도 | Top-3 추천 결과에 기대 키워드 또는 기대 분류가 포함되는지 확인 | 15/15, 100% |
| 분류 적합도 | Top-3 추천 결과 중 기대 카테고리 자원이 포함되는지 확인 | 14/15, 93% |
| 근거 충실도 예비 점검 | 추천 카드 45개가 URL·신청방법 등 DB 근거 필드를 포함하는지 확인 | 45/45, 100% |
| 추가 검토 문항 | Top-3 적합 기준 미통과 문항 | 0개 |

## 해석

위 결과는 정식 생성형 AI 답변 평가가 아니라, DB 기반 검색·추천 알고리즘의 1차 예비 평가이다. 현재 MVP에서는 추천 판단을 LLM에 맡기지 않고 DB 기반 검색 엔진이 수행하며, CLOVA Studio는 선택된 추천 카드와 근거 자원을 쉬운 한국어로 설명하는 보조 레이어로 제한한다.

따라서 본 단계에서 수행 완료로 볼 수 있는 것은 `Top-3 적합도`, `분류 적합도`, `근거 충실도 예비 점검`이다. 지도교수님이 제안한 ROUGE, METEOR, BERTScore, FactScore/Faithfulness 기반의 정량 평가는 생성 답변 로그와 정답 예시 답변이 축적된 뒤 별도 Colab/관리자 평가 환경에서 수행하는 후속 과제로 분리하였다.

## 아직 수행 전인 평가

| 지표 | 현 상태 | 필요한 추가 자료 |
|---|---|---|
| ROUGE | 설계 반영, 실행 전 | 기준 답변(reference answer)과 생성 답변 |
| METEOR | 설계 반영, 실행 전 | 기준 답변과 생성 답변 |
| BERTScore | 설계 반영, 실행 전 | 기준 답변과 생성 답변, 임베딩 모델 실행 환경 |
| FactScore/Faithfulness | 예비 점검 구현, 정식 자동화 전 | 문장 단위 생성 답변, 검색 context, 근거 매칭 규칙 |
| 환각율 | 예비 점검 구현, 정식 자동화 전 | DB 외 전화번호·URL·자격조건 생성 여부 로그 |
